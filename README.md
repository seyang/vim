vim
===

1. Intall Vundle

  git clone https://github.com/gmarik/Vundle.vim.git ~/.vim/bundle/Vundle.vim



2. Set vimrc

  1) cd ~

  2) wget https://github.com/seyang/vim/blob/master/.vimrc

  3) vi ~/.vimrc

  4) :PluginInstall



3. Set .bash_profile

  1) vi ~/.bash_profile
  
  2) add [export TERM='xterm-256color']
